1. In Admin Center, click the People icon () in the sidebar, then select Team > Team members.
  2. At the top of the page click add user.
  3. Enter the user's Name and Email.
  4. Choose Staff member for the User type.
  5. Select a Role.
  6. Click Add.
    Note: If you receive an error, you might need to purchase more agent seats. Depending on your account, owners can add agent seats a few different ways (see Adding agent seats to your subscription).
  7. In the user's profile, enter user information (see About the user options below), set agent permission (see About agent privileges and ticket access), and then change the agent's default group, if needed (see Changing the default group for your account or an agent).

    On Enterprise plans, agent permissions are not set in the user profile, but are determined by the custom role you select for the agent (see Creating custom roles).

  8. If you want to add additional contact information, click Add contact, select the type of contact, then enter the contact info in the field that appears. Repeat as needed.

    The new team member is saved automatically, so when you are finished you can simply navigate away. The new team member receives the welcome email and verification link to choose a password and sign in for the first time.
