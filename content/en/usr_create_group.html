1. In Admin Center, click the People icon () in the sidebar, then select Team > Groups.
  2. Click Add group.
  3. Enter a group name.
  4. Select the agents you want to add to the group.
  5. If you want this group to be the default group that all new agents are added to, click Make default group.

    For more information, see Changing the default group for your account or an agent.

  6. Click Create group.
